Dark mode

accordion one

This project work is concentrated on the development of a biometrically controlled fingerprint door lock, in order to improve security measures in private offices and simplified access to doors in homes, and offices. It involves the design, fabrication and testing of the system. The system is designed based on the principles of biometric identification; servo rotation and jam lock closing operation. The basic components employed in the design of the system includes the Arduino UNO microprocessor board. Servo motor, a jam lock, power supply and a fingerprint sensor module
